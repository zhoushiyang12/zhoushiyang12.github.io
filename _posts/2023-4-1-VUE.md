# Vue

和Jqurey一样通过HTML中引入vue.js的包进行应用。
Vue对象和html中的元素是一一对应的

    new Vue({
        el:'#div', #使用CSS中的选择器，选择html元素
        data:{
            name:'aaa', #VUE对象中可以使用的变量
        }
    })

### 数据绑定：
单向：v-bind
双向：v-model,可以简写成v-model eg:
    <input type=“name='dad' ,v-model:value='dad'”>

data的两种写法：+ 1、对象式，+ 2、函数式
